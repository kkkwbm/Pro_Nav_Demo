# TEBAR Demo (SerwisPro)

  A React-based demo frontend for **Pro Nav** — a professional service management platform designed for heating systems, gas boilers, and HVAC equipment maintenance
  companies.

  ## Features

  - **Interactive Map** — Google Maps integration showing all customer locations with filtering by device type, inspection status, and service confirmation
  - **Customer Management** — Full CRUD for customers, devices, service history, and device images
  - **Service Calendar** — Google Calendar integration for scheduling and tracking inspections
  - **SMS Notifications** — Send service reminders and manage SMS templates
  - **Admin Dashboard** — User management, reports, SMS history, and external service health checks
  - **Demo Mode** — Fully functional UI with mock data for testing and presentations

  ## Tech Stack

  | Category | Technology |
  |----------|------------|
  | Framework | React 19.1 |
  | Build Tool | Vite 7.0 |
  | UI Library | Material-UI 7.2 |
  | State | Redux Toolkit + React Context |
  | Routing | React Router DOM 7.8 |
  | Forms | React Hook Form + Yup |
  | Maps | @react-google-maps/api |
  | HTTP | Axios |
